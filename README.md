# Rabot

A full-stack AI-powered robot system combining a web dashboard, backend API, AI vision microservice, and Raspberry Pi motor/sensor client.

## Repository Structure

This monorepo will contain:
- `frontend/` – Web dashboard for robot control and live video
- `backend/` – Cloud API and real-time command queue
- `vision-service/` – AI model inference (object/gesture detection)
- `robot-client/` – Raspberry Pi control scripts (motors, sensors, streaming)
- `infra/` – Docker, CI/CD, deployment configuration

## Status

Project setup in progress. Components will be added step-by-step.
